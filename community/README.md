## 概览

本目录包含 OpenMAIC 开源社区的相关资源。

```
community/

```

## 飞书社区群

OpenMAIC 使用飞书（Lark）作为主要社区沟通渠道。

### 加入方式

扫描 `feishu.md` 中的二维码加入 OpenMAIC 开源社区飞书群。

### 社区用途

- 问题解答和技术支持
- 功能建议和讨论
- 发布更新和公告
- 用户交流分享
